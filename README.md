# pygame2

간단한 **Galaga 스타일 슈팅 게임**을 `pygame`으로 구현한 프로젝트입니다.  
플레이어는 좌우 이동과 발사를 통해 적을 제거하고, 적을 맞힐수록 멀티샷이 강화됩니다.

## 주요 특징

- 스크롤되는 우주 배경 연출
- 플레이어/적/총알을 `pygame.sprite`로 관리
- 적을 맞출 때마다 멀티샷 단계 업그레이드(최대 5발)
- 점수, 목숨, 멀티샷 상태 UI 표시
- 모든 적 처치 시 다음 웨이브 자동 생성
- 게임 오버 후 `R` 키로 재시작 가능

## 조작법

- `←` / `→` : 좌우 이동
- `Space` : 발사
- `R` : 게임 오버 시 재시작

## 실행 방법

1. 의존성 설치

```bash
pip install pygame
```

2. 게임 실행

```bash
python game.py
```

## 게임 규칙 요약

- 시작 목숨은 3입니다.
- 적과 충돌하면 목숨이 1 감소합니다.
- 적을 맞출 때마다 점수 +10, 멀티샷이 1단계 증가합니다.
- 멀티샷은 최대 5단계까지 증가합니다.
- 적이 화면 아래로 내려오거나 목숨이 0이 되면 게임 오버입니다.

## 파일 구조

- `game.py` : 게임 전체 로직(오브젝트 정의, 충돌 처리, 렌더링, 루프)
